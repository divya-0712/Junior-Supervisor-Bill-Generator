Junior-Supervisor-Bill-Generator
A full-stack web app to generate Junior Supervisor exam duty bills as .xlsx and PDF, built with Node.js, Express, and ExcelJS.

A web-based bill generation tool for Junior Supervisors at Savitribai Phule Pune University. Enter the supervisor's name, working dates, and the app automatically calculates the total remuneration (Days × ₹200), fills the official university bill template, and generates a downloadable Excel (.xlsx) and PDF file — preserving the original template format exactly.

Tech Stack: Node.js · Express · ExcelJS · LibreOffice (headless) · HTML/CSS/JS

Run Commends 
-cd backend 
-npm install 
-node server.js
